We are pleased to launch the refreshed Divisional website. The new site is built on a fast, accessible foundation and can be updated in two ways:

1. **By committing a Markdown file** to the GitHub repository — perfect for technical contributors.
2. **By using the friendly editor** at [`/admin/`](/admin/) — no Git knowledge required.

Notices, event details and Conclave information will appear here in due course. Suggestions and content contributions are warmly welcomed; please write to the Divisional Recorder.
